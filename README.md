# mk-note-website

Markdown Notes 官网，基于 Vite + React 构建，部署在 GitHub Pages。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

访问地址：`https://luanluuu.github.io/mk-note-website/`

## 下载链接

下载按钮指向 [mk-note/releases](https://github.com/luanluuu/mk-note/releases/latest)。
发布桌面应用时，请把 `.dmg` 和 `.exe` 上传到该 Release 页面。
